# Wordle Solver

A **Wordle solver** that calculates the best guesses.

- takes the history into consideration
- uses entropy to calculate the best guesses
- has a small html file that you can play with and try it

---

### Requirements

You need **Python 3.13 or higher** to run this.

```bash
pip install -r requirements.txt
```

---

### Usage

1. Run `solver.py` once to generate the caches and to simulate the strategy.
2. Run `main.py` and go to **localhost:5000**
